Negrito / Destaque

Nesta frase, temos um termo em negrito usando a tag <b> (não semântica)

Nesta frase, temos um termo em destaque usando a tag <strong> (semântica)

Itálico / Ênfase

Nesta frase, temos um termo em itálico usando a tag <i> (não semântica)

Nesta frase, temos um termo em ênfase usango a tag <em> (semântica)

Marcadores de Texto

Podemos criar também um texto marcado com a tag <mark>. Normalmente, eles ficam na cor amarela.

Mas podemos trocar a cor do marca texto usando o parâmetro "style" dentro da tag mark: desse jeito.

Note, porém, que ao utilizar a tag novamente, ela não herda as cores. Para isso, é preciso colocar o style na tag <head>, responsável pelas configurações do site.

Texto Grande e Pequeno

Este é um texto grande e este é um texto pequeno. A tag <big> caiu em desuso, pois podemos usar CSS para aumentar um texto. Já a tag <small> ainda está ativa pois possui um sentido (notas de rodapé, por exemplo).

Texto riscado e sublinhado

Podemos marcar um texto com a tag <del>.

Podemos sublinhar um texto com a tag <u> (não semântica).

Podemos sublinhar um texto com a tag <ins> (semântica).

Texto sobrescrito e subescrito

Usa-se a tag <sup> para textos sobrescritos como: hipotenusa2 = cat. adj.2 + cat. opos.2.

Usa-se a tag <sub> para textos como: H2O.
